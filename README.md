## ✨ Features

- **Typing animation** — rotating roles in the hero section
- **Floating tech icons** — interactive icons with profile photo repulsion effect
- **Scroll-triggered animations** — sections animate in *and* out smoothly as you scroll
- **Skills section** — flip-card entrance/exit animations per skill card
- **Portfolio tabs** — Projects, Certificates, and Experience with animated transitions
- **Certificate modal** — click any certificate to view it full-screen
- **Contact form** — with toast notification feedback
- **Navbar highlight** — smooth sliding indicator that follows active section
- **Marquee dividers** — scroll-reactive direction-aware banners
- **Logo bounce** — click the navbar logo for a surprise

## 🚀 Getting Started

No build tools, no dependencies. Just clone and open.

```bash
git clone https://github.com/Finsa-SC/portfolio.git
cd portfolio
```

Then open `index.html` in your browser — or use a local dev server:

```bash
# Python
python -m http.server 8080

# Node (npx)
npx serve .
```
